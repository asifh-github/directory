THIS IS A CORPORATE POSITION

The Data Analytics Engineer will be responsible for building and maintaining the data layer for Papa John’s analytics stack, top to bottom. He/she will work closely with multiple disciplines from data engineering to data analytics and visualization across all stages of data maturity for the purpose of delivering robust Business Intelligence solutions. This role will consider software engineering best practices including version control, automated testing, documentation, code review and continuous integration, as essential to any data stack. The Data Analytics Engineer will build data pipelines, craft data visualizations, and assist with a wide range of teams to interpret large quantity of data generated by Papa John’s existing systems. He/she will be responsible for everything ranging from collecting data, reporting facts, investigative analysis, building machine learning and computer vision models.

  • Responsible for the analytics data warehouse, implementing analytical database design principles and establishing business process best practices.

  • Develop and support data pipelines, data lake, and data warehouse solution along with finding ways to unlock value from data to support Papa John’s business decisions.

  • Provide clean data sets / products to product owners, analytics/business intelligence teams and end users.

  • Model data to empower and create straight forward data APIs and allow front end developers to easily wire up the visual components.

  • Maintain and monitor data pipeline services and BI-platforms.

  • Leverage engineering methods and best practices to deliver well-defined, tested, documented, and code-reviewed shared data sets.

  • Identify, design, and implement internal process improvements (i.e., automating manual processes, optimizing data delivery, re-designing infrastructure for greater scalability, etc.).

  • Cross-functionally supports analytical requests for both internal and external stakeholders.

  • Create and maintain optimal data pipeline architecture.

  • Assemble large, complex data sets that meet functional / non-functional business requirements.

  • Build the infrastructure required for optimal extraction, transformation, and loading of data from a wide variety of data sources using ingestion technics and AWS technologies.

  • Create data tools for analytics and data scientist team members that assist them in building and optimizing Papa John’s product into an innovative industry leader.

  • Work with data and analytics experts to strive for greater functionality in our data systems.

  • Establish scalable, efficient, automated processes for large scale data analyses, model development, validation, and implementation.

  • Solve complex data problems to deliver insights that helps our business to achieve their goals.

  • Create data products for analytics and data scientist team members to improve their productivity.
