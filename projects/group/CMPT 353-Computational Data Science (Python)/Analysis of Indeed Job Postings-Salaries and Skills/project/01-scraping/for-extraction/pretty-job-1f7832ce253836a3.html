THE TEAM:

The Data Center GPU and Accelerated Processing Team delivers AMD products for Data Center, Machine Learning, and HPC (High Performance Computing). The DC GPU Validation team is responsible for validating various GPU/APU features, platforms, and technologies prior to the production.


JOB PURPOSE:

You will be a key member of the Data Center GPU Validation team responsible for validating and debugging graphics features of AMD GPU/APU data center products.


KEY RESPONSIBILITIES:

  • Create and execute test procedures to validate the design of various GPU domains and features
  • Debug failures in collaboration with various engineering teams to deliver quality products to customers
  • Plan and report validation progress to program leads
  • Develop and maintain automation framework for validation test cases using Ruby, Python and other programming languages

WHAT YOU WILL LEARN

  • Thorough understanding of post-silicon engineering phases
  • Hands-on system validation and debug techniques
  • Strong understanding of GPU and accelerated compute technologies
  • Automation framework and scripting development

PREFERRED SKILLS AND EXPERIENCE:

  • Knowledge of hardware architecture, memory, data fabric, interconnects, high-speed interfaces, and/or system management
  • Knowledge of software architecture (BIOS, firmware, OS and driver)
  • Experience with Linux and Windows operating systems
  • Knowledge of scripting languages (Python/Ruby) and/or programming languages (C/C++)
  • Hands-on experience with laboratory equipment (multimeters, digital oscilloscopes, logic analyzer, etc.)
  • Good understanding of circuit design, schematics, and other technical data sheets
  • Excellent verbal and written communication skills

ACADEMIC CREDENTIALS:

  • BS/MS in Computer Engineering, Software Engineering, Computer Science or related field
