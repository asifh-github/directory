Job Description

Develop, test, and maintain data pipelines and analytical solutions using Python, Tableau, SQL in an Agile/Scrum environment. Implement AL/ML, and big-data techniques to cleanse, organize, and transform data. Build and maintain complex data models that enable stakeholders to make key business decisions. Perform quantitative and qualitative data analysis to produce dashboards, reports, and assessments to help make informed and data driven business decision. Provide technical expertise in data storage structures, data mining, and data cleansing. Create dashboard reports, business intelligence reports and visualizations using Tableau. Develop continuous integration and continuous deployment pipelines for data solutions that include automated unit & integration testing. Define, prepare, execute, and implement data validation and unit testing methods to ensure data quality.
